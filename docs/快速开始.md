# 快速开始

本文档介绍如何快速安装和运行 PANDA 系统。

## 环境要求

| 软件 | 版本要求 |
|------|----------|
| Python | 3.12+ |
| Node.js | 20+ |
| MySQL | 8.0+ |
| Redis | 5.0+ |

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/BAIKEMARK/PANDA.git
cd PANDA
```

### 2. 初始化数据库

```bash
# 登录 MySQL
mysql -u root -p

# 执行初始化脚本
CREATE DATABASE panda CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE panda;
SOURCE docs/项目文档/resources/panda.sql;
```

### 3. 启动 Redis

```bash
# Windows
redis-server

# Linux/Mac
sudo service redis-server start
```

### 4. 配置后端

```bash
cd backend

# 创建环境配置文件
cp .env.example .env

# 编辑 .env 文件，配置以下信息：
# - 数据库连接信息
# - JWT 密钥
# - AI API Key (通义千问)
# - Redis 连接信息
```

```bash
# 安装依赖
pip install -r requirements.txt

# 启动后端服务
python start.py
```

### 5. 配置前端

```bash
cd frontend

# 创建环境配置文件
cp .env.example .env

# 编辑 .env 文件，配置后端 API 地址
# VITE_API_URL=http://localhost:8000/api

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 常见问题

### 数据库连接失败

检查 `backend/.env` 中的数据库配置是否正确。

### Redis 连接失败

运行 `redis-cli ping` 检查 Redis 服务是否正常运行。

### AI 调用失败

检查 `.env` 中的 `AI_TEXT_KEY` 是否为有效的通义千问 API Key。

### 前端无法连接后端

检查 `frontend/.env` 中的 `VITE_API_URL` 是否正确配置。
