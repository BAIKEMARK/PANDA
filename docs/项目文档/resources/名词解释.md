# 名词解释

#### 一、围产期与围产期抑郁（依据《2021围产期抑郁症筛查与诊治专家共识》）

| **术语名称**       | **英文缩写/全称**                  | **定义与解释**                                               |
| ------------------ | ---------------------------------- | ------------------------------------------------------------ |
| **围产期**         | **Perinatal Period**               | 依据《2021专家共识》，临床筛查与管理的范围通常涵盖**妊娠期至产后1年**。虽然DSM-5在诊断上将其限定为“围产期发作”需在孕期或产后4周内，但ICD-10及临床实践多涵盖至产后6周甚至1年，以确保长周期的心理健康管理。 |
| **围产期抑郁症**   | **Perinatal Depression (PND)**     | 指在妊娠期及分娩后（通常至产后1年内）出现的抑郁症状。它是妊娠期及产褥期常见的精神障碍。核心症状包括心境低落、兴趣丧失、精力减退（“三低”），严重者可伴有自杀意念或杀婴倾向。 |
| **围产期抑郁筛查** | **Perinatal Depression Screening** | 指通过标准化工具对孕产妇进行心理健康评估的过程。《专家共识》建议对**所有**孕产妇进行至少1次筛查。对于具有高危因素（如既往史、家庭暴力等）的人群，应增加筛查频次。 |

#### 二、筛查工具与时点

| **术语名称**           | **英文缩写/全称**                               | **定义与解释**                                               |
| ---------------------- | ----------------------------------------------- | ------------------------------------------------------------ |
| **爱丁堡产后抑郁量表** | **Edinburgh Postnatal Depression Scale (EPDS)** | 围产期抑郁筛查的“金标准”工具。共10个条目，总分30分。《专家共识》推荐：**总分≥9分**为筛查阳性，提示需进一步临床评估；**总分≥13分**提示主要抑郁障碍的可能性极大，建议转诊精神科。 |
| **患者健康问卷‑9项**   | **Patient Health Questionnaire-9 (PHQ-9)**      | 通用的抑郁自评量表，亦适用于围产期。评分分级为：0-4分正常；**5-9分轻度**；**10-14分中度**；15-19分中重度；20-27分重度。常用于辅助评估抑郁的严重程度。 |
| **筛查时点**           | **Screening Time Points**                       | 《专家共识》推荐的全程筛查时机包括：**首次产检（孕早期）**、妊娠中期（30-32周）及**产后访视（产后42天）**。对于高危孕产妇，建议贯穿整个围产期进行动态监测。 |

#### 三、治疗与干预原则

| **术语名称**       | **英文缩写/全称**            | **定义与解释**                                               |
| ------------------ | ---------------------------- | ------------------------------------------------------------ |
| **结构化心理治疗** | **Structured Psychotherapy** | 对于轻中度PND，《专家共识》推荐作为**一线首选治疗**。主要包括认知行为治疗（CBT）和人际心理治疗（IPT）。本研究的THP即属于此类方法的简化应用。 |
| **药物治疗**       | **Pharmacotherapy**          | 对于重度或心理治疗无效者，建议转诊精神专科并考虑药物治疗。常用**SSRI类药物**（如舍曲林、氟西汀等），需在医生指导下权衡母婴获益与风险后使用。 |
| **多学科协作诊疗** | **MDT**                      | 强调产科、精神科、心理科、护理团队及社工的跨学科合作。建立“筛查-评估-干预-转诊”的闭环管理机制，确保母婴安全。 |

#### 四、理论模型与系统建设

| **术语名称**            | **英文缩写/全称**                    | **定义与解释**                                               |
| ----------------------- | ------------------------------------ | ------------------------------------------------------------ |
| **健康思维计划**        | **Thinking Healthy Programme (THP)** | 由**世界卫生组织（WHO）\**开发并推荐的、基于\**认知行为治疗（CBT）\**的围产期心理干预工具。其核心原理是识别\**“不健康思维”**（Unhealthy Thinking），并用**“健康思维”**（Healthy Thinking）进行替代。该方法专为非精神科专业人员（如护士、社区工作者）设计，易于推广。 |
| **PND管理模型**         | **PND Management Model**             | 本研究基于THP理论构建的标准化护理作业框架。包含四大核心环节：风险识别（Identify）、护患沟通（Communicate）、初步干预（Intervene）及专业转介（Refer）。 |
| **PND管理智能培训系统** | **Intelligent Training System**      | 本项目开发的数字化教学平台。它将THP的干预流程转化为可视化的学习模块，并利用AI技术提供实战演练环境，旨在提升护理人员的PND管理核心胜任力。 |
| **情景化智能体模块**    | **Scenario-based Agent Module**      | 系统的核心组件。基于大语言模型（LLM）构建的**“虚拟患者”**，能模拟不同PND风险等级（如轻度焦虑至重度抑郁）及情绪反应。用于支持护理人员进行高保真的对话演练。 |
| **评估—反馈—引导—总结** | **A-F-G-S Loop**                     | 智能体与学员的交互逻辑闭环：智能体根据学员的对话进行实时**评估**（Assessment），提供即时**反馈**（Feedback），通过THP原则进行**引导**（Guide），并在演练结束后生成能力**总结**（Summary）。 |

#### 五、研究方法

| **术语名称**       | **英文缩写/全称**            | **定义与解释**                                               |
| ------------------ | ---------------------------- | ------------------------------------------------------------ |
| **德尔菲专家咨询** | **Delphi Method**            | 通过多轮匿名专家函询，对PND管理模型的指标体系及智能体的话术脚本进行科学界定，使专家意见达成一致（通常共识度需>75%），确保内容的权威性。 |
| **混合研究方法**   | **Mixed Methods Research**   | 采用“定性+定量”的研究策略。前期通过**文献分析与访谈**（定性）构建模型，后期通过**问卷调查与实验研究**（定量）验证系统的应用效果。 |
| **前后对照研究**   | **Pre-test/Post-test Study** | 一种准实验设计。通过对比护理人员在使用智能培训系统**前**与**后**的知识（Knowledge）、技能（Skill）及自我效能感（Self-efficacy）得分变化，量化评估系统的培训成效。 |
