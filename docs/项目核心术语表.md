### 项目核心术语表
根据您新增的要求，结合围产期抑郁筛查的临床规范与项目背景，为您更新并扩充了项目核心术语表。

| 术语名称                                                     | 英文缩写/全称                                           | 定义与解释                                                   |
| ------------------------------------------------------------ | ------------------------------------------------------- | ------------------------------------------------------------ |
| **一、围产期与围产期抑郁（依据《2021围产期抑郁症筛查与诊治专家共识》）** |                                                         |                                                              |
| **围产期**                                                   | Perinatal Period                                        | 依据专家共识，指**妊娠期及产褥期**的连续过程。DSM‑5将妊娠期或产后4周内出现的抑郁障碍统称为重度抑郁障碍伴围产期发作；ICD‑10将产后6周内的抑郁发作归入“与产褥期有关的轻度精神和行为障碍”。本共识推荐采用DSM‑5的定义。 |
| **围产期抑郁症**                                             | Perinatal Depression (PND)                              | 指在**妊娠期及分娩后或流产后**出现的抑郁症状，包括产前抑郁症和产后抑郁症。是妊娠期及产褥期常见的精神障碍之一。其核心症状包括情感低落、兴趣及愉快感缺乏、精力或体力下降，也可能表现为易激惹，严重者可产生自杀意念及行为。 |
| **围产期抑郁筛查**                                           | Perinatal Depression Screening                          | 指通过标准化、有效的筛查工具，在孕期或产后对孕产妇进行**至少1次**抑郁和焦虑水平评估的过程。专家共识建议对所有孕妇进行抑郁症筛查，尤其对有多种危险因素的孕妇可能需要多次筛查。 |
| **二、筛查工具与时点**                                       |                                                         |                                                              |
| **爱丁堡产后抑郁量表**                                       | Edinburgh Postnatal Depression Scale (EPDS)             | **最常用的围产期抑郁症筛查工具**，共10个条目，0‑3分四级评分，总分最大值30分。专家共识指出：**总分＞12分**为筛查阳性，提示孕产妇罹患围产期抑郁症的可能性大，应转精神专科明确诊断；分值越大，症状越重。许多研究使用的临界值为**≥10分**。临床发现也适用于产前抑郁症的筛查。 |
| **患者健康问卷‑9项**                                         | Patient Health Questionnaire-9 (PHQ-9)                  | 简便、有效的抑郁自评量表。量表评分分级为：**＜5分正常；5‑9分轻度；10‑14分中度；15‑19分中~重度；＞19分重度**。可用于围产期抑郁的筛查与严重程度评估。 |
| **筛查时点**                                                 | Screening Time Points                                   | 专家共识推荐：**合理的筛查时机是早期妊娠（A级推荐）**。对于有多种危险因素（如既往抑郁症病史、低收入、亲密伴侣暴力等）的孕妇，建议**多次筛查**，如在妊娠早期和晚期各筛查1次。 |
| **三、治疗与干预原则**                                       |                                                         |                                                              |
| **结构化心理治疗**                                           | Structured Psychotherapy                                | 对轻度和中度围产期抑郁症，专家共识推荐**作为一线治疗方法**，包括认知行为治疗（CBT）和人际心理治疗（IPT）等。 |
| **药物治疗**                                                 | Pharmacotherapy                                         | 对重度围产期抑郁症者，建议**转至精神专科就诊**，推荐初始治疗采用抗抑郁药物。一线药物包括选择性5‑羟色胺再摄取抑制剂（SSRI），如舍曲林、西酞普兰和艾司西酞普兰（均为B级推荐）。 |
| **多学科协作诊疗**                                           | Multidisciplinary Collaborative Diagnosis and Treatment | 专家共识指出，应遵循以**综合、全程、分级、多学科协作诊疗**为治疗原则，保障孕产妇及胎儿安全。产科应与精神科建立合作机制，及时转诊与联合管理。 |
| **四、理论模型与系统建设**                                   |                                                         |                                                              |
| **全程健康促进计划**                                         | Total Health Promotion (THP)                            | **本研究的核心干预框架**。强调覆盖孕前、孕期、分娩及产后的全周期，整合生理、心理、社会与环境多维度因素，进行连续性、系统性的健康促进与风险管理。本研究将THP理念转化为护理人员可掌握的PND管理思维、沟通与干预模型。 |
| **PND管理模型**                                              | PND Management Model                                    | 基于THP理论，结合Delphi专家咨询与流程建模构建的标准化管理框架，包含**识别（筛查）、沟通、初步干预及转介**等关键环节，为护理人员提供可遵循的操作逻辑。 |
| **PND管理智能培训系统**                                      | Intelligent Training System for PND Management          | 本研究的核心成果，集**知识学习、情景演练与实践指导**于一体的数字化平台。将基于THP的PND管理模型转化为可学习、可演练、可指导的培训体系。 |
| **情景化智能体模块**                                         | Scenario-based Agent Module                             | 智能培训系统的核心组成部分。利用AI技术构建的虚拟患者，能够模拟不同风险等级与反应的孕产妇，与护理人员通过自然语言进行交互，用于沉浸式情景化演练。 |
| **评估—反馈—引导—总结**                                      | Assessment-Feedback-Guide-Summary                       | 智能体与护理人员交互的核心逻辑闭环：智能体根据护理人员的操作（如使用EPDS/PHQ-9筛查、沟通话术）给予**精准反馈**，并基于THP原则提供**引导**，最后进行**总结**，形成学习闭环。 |
| **五、研究方法**                                             |                                                         |                                                              |
| **德尔菲专家咨询**                                           | Delphi Method                                           | 通过**多轮匿名专家问卷调查**，使专家意见逐步趋于一致的研究方法。本研究用于构建基于THP的PND管理模型、指标体系及智能体训练脚本，确保其科学性与权威性。 |
| **混合研究方法**                                             | Mixed Methods Research                                  | 结合**定量研究**（问卷调查、前后对照实验）与**定性研究**（文献分析、半结构化访谈）的研究策略，以全面评估能力现状、构建模型并验证系统效果。 |
| **前后对照研究**                                             | Pre-test/Post-test Study                                | 实验设计方法，通过对比护理人员在使用智能培训系统**前后**的知识掌握度、技能熟练度及自我效能感，以验证系统的有效性与应用价值。 |
