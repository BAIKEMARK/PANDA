# 代理配置使用指南

## 概述

本项目已集成代理支持，可以方便地通过代理访问 Google API 等需要代理的服务。

## 配置方式

### 1. 环境变量配置

在 `app/.env` 文件中添加代理配置：

```env
# 代理配置
HTTP_PROXY=http://127.0.0.1:7897
HTTPS_PROXY=http://127.0.0.1:7897
```

### 2. 配置说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `HTTP_PROXY` | HTTP 代理地址 | `http://127.0.0.1:7897` |
| `HTTPS_PROXY` | HTTPS 代理地址 | `http://127.0.0.1:7897` |

### 3. 代理格式

支持以下代理格式：
- HTTP 代理: `http://host:port`
- SOCKS 代理: `socks5://host:port`
- 带认证: `http://username:password@host:port`

## 使用方法

### 方式一：自动应用（推荐）

代理会在应用启动时自动设置，所有 HTTP 请求库（requests、httpx、aiohttp）会自动使用代理。

```python
# main.py
from proxy_utils import setup_proxy

# 应用启动时自动执行
setup_proxy()
```

### 方式二：手动获取代理配置

#### requests 库

```python
from proxy_utils import get_proxies
import requests

proxies = get_proxies()
response = requests.get("https://api.google.com", proxies=proxies)
```

#### httpx 库

```python
from proxy_utils import get_httpx_proxy
import httpx

proxies = get_httpx_proxy()
client = httpx.Client(proxies=proxies)
response = client.get("https://api.google.com")
```

#### aiohttp 库

```python
from proxy_utils import get_proxies
import aiohttp

proxies = get_proxies()
async with aiohttp.ClientSession() as session:
    async with session.get("https://api.google.com", proxy=proxies.get("https")) as response:
        data = await response.json()
```

## 实际应用示例

### Google Custom Search API

```python
from proxy_utils import get_proxies
import requests

def google_search(query: str):
    """使用代理调用 Google 搜索 API"""
    url = "https://www.googleapis.com/customsearch/v1"

    params = {
        "key": settings.GOOGLE_API_KEY,
        "cx": settings.GOOGLE_CSE_ID,
        "q": query,
        "num": 10
    }

    proxies = get_proxies()
    response = requests.get(url, params=params, proxies=proxies)

    return response.json()
```

### OpenAI API (如果需要代理)

```python
from proxy_utils import get_httpx_proxy
from openai import OpenAI

proxies = get_httpx_proxy()
client = OpenAI(
    api_key=settings.AI_TEXT_KEY,
    http_client=httpx.Client(proxies=proxies)
)

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello"}]
)
```

## 代理常见问题

### 1. 代理连接失败

**错误**: `requests.exceptions.ProxyError`

**解决方案**:
- 检查代理服务是否运行（如 Clash、V2Ray）
- 确认代理端口是否正确
- 检查防火墙设置

### 2. SSL 证书验证失败

**错误**: `SSLError: [SSL: CERTIFICATE_VERIFY_FAILED]`

**解决方案**:

```python
# 方式一：禁用 SSL 验证（不推荐，仅用于测试）
response = requests.get(url, proxies=proxies, verify=False)

# 方式二：使用系统证书
import certifi
response = requests.get(url, proxies=proxies, verify=certifi.where())
```

### 3. 不同服务使用不同代理

如果某些服务不需要代理，可以在请求时单独配置：

```python
# 本地服务不使用代理
response = requests.get("http://localhost:8000/api/health")

# 外部服务使用代理
response = requests.get("https://api.google.com", proxies=get_proxies())
```

## 禁用代理

如果不需要使用代理，在 `.env` 文件中注释掉或删除代理配置：

```env
 # HTTP_PROXY=http://127.0.0.1:7897
 # HTTPS_PROXY=http://127.0.0.1:7897
```

或设置为空：

```env
HTTP_PROXY=
HTTPS_PROXY=
```
