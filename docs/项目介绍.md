# 项目二：基于THP的围产期抑郁管理智能培训系统开发与应用

## 项目简介：
国家层面高度重视围产期心理健康问题，将其纳入生育支持与妇幼健康服务体系整体布局，并推动"生育友好型医院"建设，要求在孕期、分娩期及产后连续服务过程中，加强对孕产妇心理风险的识别、干预与转介。

本研究针对护理人员在围产期抑郁（PND）管理中规范化能力不足的问题，基于健康思维计划（THP），结合AI技术，开展提升护理人员PND管理能力的系统研究。

研究采用文献分析、问卷调查和半结构化访谈相结合的混合研究方法，系统评估护理人员在PND识别、沟通支持和初步干预中的能力现状与培训需求。在此基础上，通过Delphi专家咨询与流程建模，构建基于THP的护理人员PND管理模型与标准化操作流程。进一步将模型转化为智能化培训方案，开发集知识学习、情景演练与实践指导于一体的PND管理智能培训系统及核心智能体模块。通过前后对照研究，验证智能培训系统在提升护理人员PND管理相关知识、技能和自我效能方面的应用效果。

研究拟形成护理人员PND管理模型、智能培训系统原型及应用评价结果，为护理人员参与围产期心理健康管理提供可推广的技术路径与实施模式。

## 基于THP的PND管理智能培训系统与智能体构建

将基于THP的护理管理模型转化为可学习、可演练、可指导的智能培训系统，构建以智能体为核心的护理人员PND管理培训与实践支持工具。

系统主要包括：THP知识与流程管理模块；护理人员培训与学习模块；情景化智能体模块（核心）；培训评估与反馈模块。

## 智能体构建流程

1. **知识结构化**：将THP核心干预内容和护理管理流程转化为结构化知识单元，形成智能体知识库。
2. **情景脚本设计**：基于真实护理场景，设计涵盖不同PND风险情境的训练脚本，用于智能体对话与演练。
3. **对话与决策逻辑设计**：构建"评估—反馈—引导—总结"的对话流程，嵌入THP干预原则与风险控制规则。
4. **测试与优化**：组织护理人员试用系统，根据反馈对智能体交互逻辑和内容呈现进行迭代优化。
